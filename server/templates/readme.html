CCollage generates SVG collages of a Git repository's contributors.

GitHub is supported. GitLab and BitBucket support might be added eventually.

Usage

Just add the repository URL and contributors.svg to the end of the domain name, like this:

https://ccollage.vc.sg/github.com/valentine/ccollage/contributors.svg

Parameters

CCollage supports parameters in the query string to customise the generated SVG image. All parameters are optional.

For example:

https://ccollage.vc.sg/github.com/valentine/ccollage/contributors.svg?s=100&p=7&w=700
Parameter Name Description
s / size Size of the contributor avatars, in pixels. Defaults to 80px.
p / padding Padding between each contributor avatar, in pixels. Defaults to 5px.
w / width Maximum width of the generated image, in pixels. Defaults to 800px.
